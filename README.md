# 🍕 Papa_Pizzas App - Android (Java)

Aplicativo Android desenvolvido em **Java** para simular um sistema de pedidos de pizzaria. O app permite visualizar sabores, adicionar ao carrinho e realizar pedidos fictícios.

## 📱 Tecnologias Utilizadas

- Java  
- Android Studio  
- XML (Layouts)  
- Firebase Firestore
- RecyclerView, CardView 

## ⚙️ Funcionalidades

- ✅ Listagem de pizzas com nome, descrição e preço  
- ✅ Adição ao carrinho  
- ✅ Cálculo automático do valor total  
- ✅ Finalização de pedido (simulado)  
- ✅ Interface responsiva e intuitiva
